VCApi

All URIs are relative to http://gateway.neiro.ai

Method HTTP request Description
vCNewVC POST /v1/new-vc Get audio with another voice.
vCVC POST /v1/vc Get audio with another voice.
vCVCV2 POST /v2/vc Get audio with another voice.
vCVCV3 POST /v3/vc Get audio with another voice.

vCNewVC

mynaGatewayResponse vCNewVC(body)

Get audio with another voice.

Parameters

Name Type Description Notes
body vcVCRequest VC request. Send speaker id (obama, eilish, etc) and audio bytes. Bytes should be in WAV, raw for GRPC, base64 for HTTP

Return type

mynaGatewayResponse

Authorization

No authorization required

HTTP request headers

  • Content-Type: application/json
  • Accept: application/json
